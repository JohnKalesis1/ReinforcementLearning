Αρχειο για παρουσιαση εργασιας ειναι το pdf, το οποιο περιεχει ολους τους γραφους που προεκυψαν απο τα πειραματα, 
συνοδευωμενο απο τα εκτενη συμπερασματα μου πανω στους αλγοριθμους και πανω στα αποτελεσματα των πειραματων.

Οι κωδικες βρισκονται σε Code directory, οπου και εχουν αρκετο σχολιασμο που εξηγουν τις σχεδιαστικες επιλογες.
Στο ιδιο directory βρισκονται και δυο εκπαιδευμενα μοντελα για dqn και ddqn, τα οποια αποθηκευτηκαν σωστα και ειναι ετοιμα για χρηση.

Αν επιχειρησετε να τα τρεξετε το agent.test() για dqn και ddqn σιγουρευτειτε οτι εκτελειτε το προγραμμα μεσα στο directory Code, ετσι ωστε να υπαρχει ορατοτητα στα αποθηκευμενα models.

Αξιζει να αναφερθει οτι λογω μεγαλου φορτου εργασιων απο αλλα μαθηματα του 5ου εξαμηνου δεν καταφερα να κανω οσα πειραματα ηθελα να κανω, 
κυριως δηλαδη να πειραματιστω και σε αλλα περιβολλοντα εκτος του Cartpole τα οποια πιστευω θα χαν και περρισοτερο ενδιαφερον.
Παρολο αυτα θεωρω τα πειραματα που γιναν στο Cartpole και η υλοποιηση των μοντελων/αλγοριθμων περιειχε ιδιαιτερο ενδιαφερον ως προς τα αποτελεσματα και πειραματισμο με παραπαμετρους.
